# Gruppierung

## Gruppierung

Einteilung der Daten in Gruppen / Kategorien und berechnen von Werten auf deren Basis

## Gruppierung

Beispiel: Durchschnittswerte der Iris-Daten basierend auf dem Namen der Art

```py
iris.groupby(iris.name).mean()
```
