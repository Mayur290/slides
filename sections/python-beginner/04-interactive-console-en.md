# Interactive Python console

## Interactive Python console

options:

- local installation
- online notebooks (Jupyter)

## Local Python console

launching the Python console / shell:

- command `python` in the command prompt
- from the start menu (e.g. _Python 3.9_)

quitting:

```py
exit()
```
